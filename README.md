This project implements a simplified IT Service Management (ITSM) solution
in ServiceNow for a watch manufacturing and retail organization.

The goal of the project is to simulate how IT teams in a real enterprise
handle incidents, service-level commitments, and change approvals
using ServiceNow.

The implementation focuses on CSA-level configurations and best practices,
including assignment groups, roles, automation, and workflows.

Modules implemented:
- Incident Management
- Change Management
- SLA enforcement
- Role-based responsibility using groups and roles

## ServiceNow Components Used
- Business Rules
- Flow Designer
- SLA Definitions
- Assignment Groups
- Users & Roles
- Reports & Dashboards
